# Aeroscript Frontend

React + Vite application for the Aeroscript collaborative drawing platform.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:
```
VITE_API_URL=http://localhost:5002
VITE_SOCKET_URL=http://localhost:5002
VITE_GOOGLE_CLIENT_ID=your-google-client-id.apps.googleusercontent.com
```
